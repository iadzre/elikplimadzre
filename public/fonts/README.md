# Fonts Folder

## Gazzetta Font Files Required

To use the Gazzetta font locally, please add the following font files to this folder:

### Required Files:
- `Gazzetta-Regular.woff2` (preferred format)
- `Gazzetta-Regular.woff` (fallback)
- `Gazzetta-Regular.ttf` (fallback)
- `Gazzetta-Bold.woff2` (preferred format)
- `Gazzetta-Bold.woff` (fallback)
- `Gazzetta-Bold.ttf` (fallback)

### How to Get the Font Files:

1. **From CDN Fonts:**
   - Visit: https://www.cdnfonts.com/gazzetta.font
   - Download the font files
   - Extract and copy the .woff2, .woff, or .ttf files to this folder

2. **From MyFonts:**
   - Visit: https://www.myfonts.com/collections/gazzetta-font-tipotype/
   - Purchase/download the font
   - Extract the web font files (.woff2, .woff, .ttf)

3. **If you already have the font:**
   - Copy the font files directly to this folder
   - Ensure they are named exactly as listed above

### Font Usage:

The font is already configured in `css/styles.css` and can be used with:
- Class: `.gazzetta` (regular weight)
- Class: `.gazzetta-bold` (bold weight)

The about page is already set up to use the local font once the files are added.
